## 알고리즘
- 컴퓨터가 어떤 일을 수행하기 위한 단계적 방법

# Tree
- 비선형 구조, 1:n 관계, 계층관계

### 용어
- 루트 : 노드 중 최상위 노드
- 서브트리 : 하나의 큰 트리를 구성하는 왼쪽과 오른쪽의 작은 트리
- 단말노드 : 맨 마지막 노드
- 차수 : 노드에 연결된 자식 노드의 수 (트리의 차수 : 트리를 구성하는 노드의 차수 중 가장 큰 값)

## 이진트리
- 정의 : 모든 노드가 자식노드를 최대 2개까지만 가질 수 있는 트리로, 모든 코드들이 2개의 서브트리를 갖는 특별한 형태
- 레벨 i 에서 노드의 최대 개수는 2**i 개
- 높이가 h인 이진트리가 가질 수 있는 노드의 최소 개수는 (h+1)개, 최대 개수는 (2**(h+1) -1) 개
- 종류
```
완전 이진트리 : 높이가 h, 노드 수가 n -> 노드번호 1번부터 n번까지 빈자리가 없는 이진트리. 현재 노드가 n번일때 왼쪽 자식노드는 n*2, 오른쪽은 n*2+1, 부모노드는 n//2
포화 이진트리 : 모든 레벨에 노드가 포화상태인 꽉 찬 트리
편향 이진트리 
```

### 순회
- 전위순회 : VLR (왼쪽점)
- 중위순회 : LVR (아래쪽 점)
- 후위순회 : LRV (오른쪽 점)

### Binary search tree
- 탐색작업을 효육적으로 하기 위한 자료구조
- 모든 원소는 서로 다른 유일한 키를 가짐
- 왼쪽 key < 루트 key < 오른쪽 key
- 중위 순회시 오름차순 정렬 값 얻을 수 있음

### Heap
: 완전 이진 트리에 있는 노드 중 key 값이 가장 큰 노드 또는 가장 작은 노드를 찾기 위해 만든 자료구조
